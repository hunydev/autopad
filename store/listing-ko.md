# AutoPad — Microsoft Store 목록 (한국어)

## 설명

AutoPad은 복사한 텍스트, 이미지, 파일을 실시간으로 감지하고 미리보기와 함께 토스트 알림을 보여주는 Windows 클립보드 유틸리티입니다.

빠르게 수정이 필요하신가요? 토스트 팝업에서 바로 내장 편집기를 열어보세요. 오타 수정, 공백 정리, 이미지 주석 추가 후 클립보드로 바로 복사할 수 있습니다. 별도 앱이 필요 없습니다.

주요 기능:
• 텍스트, 이미지, 파일 복사를 실시간 모니터링
• 콘텐츠 미리보기, 글자 수, 바이트 크기를 포함한 토스트 알림
• 즉시 복사 가능한 내장 텍스트 편집기
• 이미지 마크업: 펜, 지우개, 영역 선택 후 모자이크, 채우기, 지우기
• JavaScript 매크로 시스템 — 28개 내장 프리셋으로 텍스트 변환
• 스티커 메모 — 복사한 내용을 항상 위 창으로 고정
• 클립보드 히스토리 (검색, 재복사)
• 이미지 Base64 복사 (전체 또는 선택 영역)
• 웹 서식 콘텐츠의 HTML 소스 뷰어
• 파일/폴더 경로 감지 및 탐색기에서 바로 열기
• 토스트 투명도, 컴팩트 모드, 맞춤법 검사 옵션
• 다크 테마 UI
• 영어/한국어 인터페이스
• 시스템 트레이에서 조용히 실행

AutoPad은 가볍고, 개인정보를 존중하며(데이터가 외부로 전송되지 않음), 필요할 때까지 방해하지 않습니다.


## 이 버전의 새로운 기능

• 샌드박스 Jint 엔진 기반 JavaScript 매크로 시스템
• 28개 내장 매크로 프리셋 (마스킹, JSON, Base64, 정렬, 추출 등)
• 매크로 편집기 (테스트 실행 및 스크립트 검증)
• 매크로 관리: 추가, 편집, 삭제, 순서 변경, 프리셋
• 스티커 메모 (고정) 기능 — 텍스트/이미지를 항상 위 창으로 고정
• 스티커: 항상 위 토글, 편집 전환, 이미지 맞추기, 크기 조절
• 이미지 Base64 복사 (전체 또는 선택 영역 → data URI)
• 특수 공백 제거 드롭다운 메뉴 (삭제/공백 치환)
• 텍스트 편집기 맞춤법 검사 옵션 (설정 > 일반)
• 설정 매크로 탭 추가 (4탭: 일반/알림/히스토리/매크로)
