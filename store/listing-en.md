# AutoPad — Microsoft Store Listing (English US)

## 설명 (Description)

AutoPad is a clipboard utility for Windows that monitors everything you copy — text, images, and files — and shows a quick toast notification with a preview.

Need to make a quick edit? Open the built-in editor right from the toast popup. Fix a typo, trim whitespace, annotate an image, then copy it back. No separate app needed.

Key capabilities:
• Real-time clipboard monitoring for text, images, and file copies
• Toast notifications with content preview, character count, and byte size
• Built-in text editor with instant copy-back
• Image markup: pen, eraser, region select with mosaic, fill, and erase
• HTML source viewer for formatted web content
• File/folder path detection with quick-open in Explorer
• UTF-8 / EUC-KR encoding conversion
• Configurable toast position and duration
• Full dark theme UI
• English and Korean interface
• Runs quietly from the system tray

AutoPad is lightweight, privacy-respecting (no data leaves your machine), and stays out of your way until you need it.
