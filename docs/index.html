Features
01

Clipboard monitoring

Watches for text, image, and file copies in real-time. Runs from the system tray.

02

Toast previews

Shows a small popup with content preview, size info, and action buttons. Position it anywhere on screen.

03

Inline text editing

Open copied text in a built-in editor. Make changes and copy back with one click.

04

Image markup

Draw, erase, select regions. Apply mosaic, fill, or erase on selections. Undo with Ctrl+Z.

05

HTML source

When you copy formatted web content, view the raw HTML behind it.

06

Path detection

Recognizes file and folder paths in copied text. Open them directly in Explorer.

07

File & image monitoring

Detects file copies with thumbnail previews. Toggle image and file monitoring independently. Configurable size limit.

08

Clipboard history

Browse, search, and re-copy or edit past clipboard entries. Configurable history size (20–200).

09

Toast opacity & compact mode

Adjust toast transparency from 10% to 100%. Hover to reveal at full opacity. Compact mode shows buttons only.

10

Live settings preview

See toast changes in real-time while adjusting position, compact mode, and opacity in the Notification settings tab.

11

Encoding conversion

Switch between UTF-8 and EUC-KR for text content.

12

Dark theme

Full dark UI including titlebar and scrollbars. It's the only theme, and it looks right.

How it works
1.

Copy something anywhere on your system. AutoPad is listening.

2.

A toast notification appears with a preview and quick actions.

3.

Hit Edit to modify, then Copy & Close to write it back.

Requirements
  • Windows 10 or 11, 64-bit
  • Self-contained build — no runtime needed
  • All data stays on your machine, no network calls
