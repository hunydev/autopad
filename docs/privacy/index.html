Privacy Policy

Last updated: March 25, 2026

Overview

AutoPad is a clipboard monitoring utility that runs entirely on your local machine. It does not collect, transmit, or store any personal data externally.

Data Collection

AutoPad does not collect any data. Specifically:

  • No personal information is collected
  • No clipboard content is transmitted over the network
  • No usage analytics or telemetry data is gathered
  • No crash reports are sent externally
  • No cookies or tracking technologies are used

Local Storage

AutoPad stores a single configuration file (settings.json) in your local AppData folder. This file contains only your app preferences (toast position, language, monitoring options) and never leaves your device.

Clipboard Access

AutoPad monitors the Windows clipboard to provide its core functionality — detecting copied text, images, and files. All clipboard data is processed locally in memory and is never stored persistently or transmitted anywhere.

Network Access

AutoPad makes zero network calls. The application does not connect to the internet for any purpose, including updates, analytics, or data transmission.

Third-Party Services

AutoPad does not integrate with any third-party services, SDKs, or APIs.

Children's Privacy

AutoPad does not collect any data from anyone, including children under the age of 13.

Changes to This Policy

If this privacy policy is updated, the changes will be posted on this page with the updated date.

Contact

If you have questions about this privacy policy, you can reach the developer via GitHub Issues.
